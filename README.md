＜本データパックについて＞  
・Minecraft Java 1.21.11　マルチ(vannila推奨)サーバー用

＜導入方法について＞
1. ダウンロードしたファイルを展開
2. 中に入っているsuraimu_manhuntファイルを(サーバーフォルダ)/world/datapacks/に追加
3. 導入完了！

＜データパック使用ルール＞
1. 使用、改変、動画化などは基本許可不要です！(一言くれれば見に行きます)
2. 二次配布については許可取りをお願いします

＜マンハントの詳細ルール＞
1. ランナーは使用したネザーポータル以外での帰還禁止
2. ハンターはネザーポータルに回避不可能のトラップ(黒曜石で囲うなど)の設置禁止(出待ちは許可)
